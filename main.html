ຍີດີຕ້ອນຮັບເຂົ້າສູ່ຫ້ອງ IT

ວັນອັງຄານ ເດືອນພະຈີກ ປີ 2018


ພິ​ທີ​ຕ້ອນ​ຮັບ​ຄະ​ນະ 3 ອົງ​ການ​ຈັດ​ຕັ້ງ​ມະ​ຫາ​ຊົນ ພະ​ແນກ​ສຶກ​ສາ​ທິ​ການ ແລະ ກິ​ລາ​ແຂວງ​ຫົວ​ພັນທີ່​ມາ​ຢ້ຽມ​ຢາມ​ແລກ​ປ່ຽນ​ບົດ​ຮຽນ​ນຳ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ຄັ້ງ​ວັນ​ທີ 3 ສິງ​ຫາ 2018.

    ໃນ​ຕອນ​ເຊົ້າ​ຂອງວັນ​ສຸກ ວັນ​ທີ ວັນ​ທີ 3 ສິງ​ຫາ 2018 ທີ່​ສະ​ໂມ​ສອນ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ແຂວງ​ຊຽງ​ຂວາງ ໄດ້​ຈັດພິ​ທີ​ຕ້ອນ​ຮັບ​ຄະ​ນະ 3 ອົງ​ການ​ຈັດ​ຕັ້ງ​ມະ​ຫາ​ຊົນ ພະ​ແນກ​ສຶກ​ສາ​ທິ​ການ ແລະ ກິ​ລາ​ແຂວງ​ຫົວ​ພັນ ທີ່​ມາ​ຢ້ຽມ​ຢາມ​ແລກ​ປ່ຽນ​ບົດ​ຮຽນ​ນຳ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ພາຍ​ໃຕ້​ການ​ນຳ​ພາ​ຂອງ​ ທ່ານ ສົມ​ເພັດ ອິນ​ທະ​ສອນ ຮອງ​ຫົວ​ໜ້າ ພະ​ແນກ​ສຶກ​ສາ​ທິ​ການ ແລະ ກິ​ລາ​ແຂວງ​ຫົວ​ພັນ ແລະ ກ່າວ​ຕ້ອນ​ຮັບ​ຂອງ​ທ່ານ ອ​ຈ ວັນ​ນີ ຢ່າງ​ເຈຍ​ມົວ ເລ​ຂາ​ພັກ​ຮາກ​ຖານ ຜູ້​ອຳ​ນວຍ​ການ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ພ້ອມ​ການ​ເຂົ້າ​ຮ່ວມ​ຂອງ​ ທ່ານ ຄະ​ນະ​ພັກ​ຮາກ​ຖານ, ຄະ​ນະ​ສະ​ຫະ​ພັນ ແລະ ຄະ​ນະ​ຮາກ​ຖານ ຂອງ 3 ອົງ​ການ​ຈັດ​ຕັ້ງ.
    ໃນ​ກອງ​ປະ​ຊຸມ​ໄດ້​ມີ​ການ​ແລກ​ປ່ຽນ ຖອດ​ຖອນບົດ​ຮຽນ ທາງ​ດ້ານ​ການ​ຈັດ​ຕັ້ງ​ປະ​ຕິ​ບັດ​ວຽກ​ງານ​ຂອງ 3 ອົງ​ການ​ຈັດ​ຕັ້ງ​ມະ​ຫາ​ຊົນ ເຊິ່ງ​ໃນ​ເບື້ອງ​ຕົ້ນ​ໄດ້​ຮັບ​ຟັງການ​ສະຫຼຸບ​ລວມ​ໂດຍ​ຫຍໍ້ການ​ຈັດ​ຕັ້ງ​ປະ​ຕິ​ບັດ​ວຽກ​ງານ​ຂອງ 3 ອົງ​ການ​ຈັດ​ຕັ້ງ​ມະ​ຫາ​ຊົນ ຂອງວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ຈາກ ສ​ຫ ຄຳ​ວົງ ອຸ​ດົມ​ສຸກ ເລ​ຂາ ຮາກ​ຖານ​ຊາວ​ໜຸ່ມ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ຈາກ​ນັ້ນ​ຮັບ​ຟັງການ​ສະຫຼຸບ​ລວມ​ໂດຍ​ຫຍໍ້ການ​ຈັດ​ຕັ້ງ​ປະ​ຕິ​ບັດ​ວຽກ​ງານ​ຂອງ 3 ອົງ​ການ​ຈັດ​ຕັ້ງ​ມະ​ຫາ​ຊົນ ຂອງພະ​ແນກ​ສຶກ​ສາ​ທິ​ການ ແລະ ກິ​ລາ​ແຂວງ​ຫົວ​ພັນ ຈາກ ສ​ຫ ວັນ​ສີ ສຸກ​ສ​ະ​ຫວັນ ເລ​ຂາ​ຊາວ​ໜຸ່ມພະ​ແນກ​ສຶກ​ສາ​ທິ​ການ ແລະ ກິ​ລາ​ແຂວງ​ຫົວ​ພັນ.

 
  ພິ​ທີ​ເປີດ​ຮຽນ​ທິດ​ສະ​ດີ​ການ​ເມືອງ ແລະ ການ​ປົກ​ຄອງ​ລະ​ບົບ​ກໍ່​ສ້າງ​ຊັ້ນ​ຕົ້ນ ຮຸ່ນ​ທີ 1​ຂອງວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ແຂວງ​ຊຽ​ຂວາງ

   ໃນເວ​ລາ 1:00 ນາ​ທີ ຂອງວັນ​ທີ 1 ສິງ​ຫາ 2018 ຢູ່​ທີ່ສະ​ໂມ​ສອນ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ໄດ້​ຈັດ​ພິ​ທີ​ເປີດ​ຮຽນ​ທິດ​ສະ​ດີ​ການ​ເມືອງ ແລະ ການ​ປົກ​ຄອງ​ລະ​ບົບ​ກໍ່​ສ້າງ​ຊັ້ນ​ຕົ້ນ ຮຸ່ນ​ທີ 1​ຂອງວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ ແຂວງ​ຊຽ​ຂວາງ ຂຶ້ນ​ພາຍ​ໃຕ້​ການ​ເປັນ​ປະ​ທານ​ຂອງ ​ທ່ານ ບົວ​ສອນ ສີ​ນວນ​ທອງ ຄະ​ນະ​ປະ​ຈຳ​ພັກ​ແຂວງ ຮອງ​ເຈົ້າ​ແຂວງ ແຂວງ​ຊຽງ​ຂວາງ ພ້ອມ​ທັງ​ການ​ເຂົ້າ​ຮ່ວມ​ຂອງ ທ່ານ ອ​ຈ ວັນ​ນີ ຢ່າງ​ເຈຍ​ມົວ ເລ​ຂາ​ພັກ​ຮາກ​ຖານ ຜູ້​ອຳ​ນວຍ​ການ ວິ​ທະ​ຍາ​ໄລ​ຄູ​ຄັງ​ໄຂ, ທ່ານ ບຸນ​ເລື່ອນ ວັນ​ນາ​ຮັກ ຜູ້​ອຳ​ນວຍກ​ານ ໂຮງ​ຮຽນ​ການ​ເມືອງ ແລະ ການ​ປົກ​ຄອງ​ແຂວງ​ຊຽ​ຂວາງ, ແລະ ບັນ​ດາ​ແຂກ​ທີ່​ຖືກ​ເຊີນ ຕະຫຼອດ​ຮອດ​ຄູ-ອາ​ຈານ ແລະ ນັກ​ສຶກ​ສາ ລວມ​ທັງ​ໝົດ 83 ກ່​ວາ​ສະ​ຫາຍ ເຊິ່ງກອງ​ປະ​ຊຸມ​ດັ່ງ​ກ່າວ​ໄດ້​​​ຮັບ​ຟັງການແນະ​ນຳຫຼັກ​ສູດ ແລະ ວິ​ທີ​ດຳ​ເນີນ​ການ​ຈັດ​ການ​ຮຽນ-ການ​ສອນ​ ຈາກ ທ່ານ ບຸນ​ເລື່ອນ ວັນ​ນາ​ຮັກ ຜູ້​ອຳ​ນວຍກ​ານ ໂຮງ​ຮຽນ​ການ​ເມືອງ ແລະ ການ​ປົກ​ຄອງ​ແຂວງ​ຊຽ​ຂວາງ ແລະ ຍັງ​ໄດ້​ຮັບ​ຟັງ​ການ​ໂອ້​ລົມ, ມີ​ຄຳ​ຄິດ​ຄຳ​ເຫັນ ແລະ ກ່າວ​ເປີດ​ພິ​ທີ​ຢ່າງ​ເປັນ​ທາງ​ການ​ຈາກ ທ່ານ ບົວ​ສອນ ສີ​ນວນ​ທອງ ຄະ​ນະ​ປະ​ຈຳ​ພັກ​ແຂວງ ຮອງ​ເຈົ້າ​ແຂວງ ແຂວງ​ຊຽງ​ຂວາງ.
